# Phishing Awareness Training

## Overview
This project was developed as part of the CodeAlpha Cyber Security Internship.

The presentation explains:
- What phishing is
- Types of phishing attacks
- How phishing works
- Social engineering techniques
- How to identify phishing emails and fake websites
- Prevention methods and best practices
- A real-world phishing example
- Interactive quiz
